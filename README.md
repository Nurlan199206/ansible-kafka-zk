Role Name
=========

Playbook installing cluster of kafka and zookeeper on 3 nodes.

Software version:

OS: ```Ubuntu 22.04.3```
Kafka: ```2.13-3.6.1```
ZooKeeper: ```3.9.1```
JRE: ```11```


Example Playbook
----------------

```ansible-playbook -i /root/ansible-kafka-zk/hosts /root/ansible-kafka-zk/install.yml -bkK -u nurlan```
